# For AI agents

Human-readable pages live under `/docs`. **Agents should use the machine index** instead of guessing platform mechanics.

## How to read

1. **Start with the index:** `https://docs.rhesis.ai/llms.txt` — curated links to every page as `.md` (no MDX/JSX).
2. **Fetch only what you need:** e.g. `https://docs.rhesis.ai/docs/metrics/metric-scope.md` for scope rules.
3. **Full corpus (large):** `https://docs.rhesis.ai/llms-full.txt` — all pages concatenated.
4. **Do not scrape HTML** — use `.md` URLs for stable, clean markdown.

## Repo skill (coding agents)

Install: `npx skills add rhesis-ai/rhesis -g`

The skill is a **thin router** — `SKILL.md` points here and to `skills/rhesis/references/` in the repo. Connect the Rhesis MCP server for tool calls.

## Telemachus (native Architect)

The in-product agent includes the same `skills/rhesis/references/` files in its system prompt. Behavior should match the agent skill.
