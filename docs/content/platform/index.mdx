# Platform

<Callout type="default">
  **Try Rhesis Now:** Access the platform at [app.rhesis.ai](https://app.rhesis.ai) to start testing your AI applications.
</Callout>

The Rhesis platform provides comprehensive tools for testing and evaluating AI applications at scale. Build, test, and improve your AI applications with confidence using our integrated testing and evaluation platform.

<Callout type="info">
  **New to Rhesis?** Start with [Core Concepts](/getting-started/concepts)
  to understand how everything fits together, or take a look at the [Getting Started](/getting-started) guide.
</Callout>

## Where to Start

If you are just getting started, follow this path:

1. **Create a [Project](/platform/projects)** - Organize your testing work
2. **Configure [Endpoints](/platform/endpoints)** - Connect to your AI application
3. **Generate [Tests](/platform/tests-generation)** - Create test cases with AI assistance
4. **Define [Metrics](/platform/metrics)** - Set up evaluation criteria
5. **Run and Analyze** - Execute tests and review results

Already familiar? Jump to any feature below.
