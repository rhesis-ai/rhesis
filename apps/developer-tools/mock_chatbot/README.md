# Mock Chatbot

A minimal FastAPI server that mimics a chatbot. It takes a `message` and replies
with random text generated by [Faker](https://faker.readthedocs.io/).

## Start

```bash
./start.sh
```

The server listens on `http://localhost:18090`.

## Usage

```bash
curl -X POST http://localhost:18090/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "hello"}'
```
