# API Tokens

Generate and manage API tokens for programmatic access to Rhesis. You can manage them at [app.rhesis.ai/tokens](https://app.rhesis.ai/tokens).

<Callout type="default">
  **What are API Tokens?** API Tokens authenticate programmatic access to Rhesis via the REST API or Python SDK. The token is shown only once at creation — copy it immediately.
</Callout>

![Token Overview](/screenshots/rhesis-ai-tokens.png)

## Using API Tokens

### With REST API

Include the token in the Authorization header:

<CodeBlock filename="Terminal" language="bash">
  {`curl -H "Authorization: Bearer YOUR_TOKEN_HERE" \\
  https://api.rhesis.com/v1/tests`}
</CodeBlock>

### With Python SDK

Initialize the SDK with your token:

<CodeBlock
  filename="client.py"
  language="python"
>
{`from rhesis import RhesisClient

client = RhesisClient(api_token="YOUR_TOKEN_HERE")`}

</CodeBlock>

See [Python SDK](/sdk/installation) for complete usage documentation.
